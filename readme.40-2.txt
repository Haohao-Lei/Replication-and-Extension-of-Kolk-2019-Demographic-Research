
Data and source files for manuscript "Weak support for a U-shaped pattern between societal gender equality and fertility when comparing societies across time" in Demographic Research

Below is the scripts and source data files (both in Stata formate) to produce all results in the manuscript.


analysis.do - contains a script to produce all results and graphs in the table. Some brief discussion of different data choices.


gender.dta	- stata file with gender index from the VoD dataset


fertility.dta - stata file with tfr across years from HFD and the World Bank


gender index data plot.do - contains a script to produce a comparison of different gender equality indices in 2010. Contains sources for the different indices.


gender index data - contains data for gender index data.do
